Message box that works with a Delphi Service Application

If got here by following a link such as http://delphidabbler.com/tips/999 (where 999 is a number) then you were re-directed because the tip is not yet available on the new delphidabbler.com website. Go back to the delphidabbler.com tips page

Here is a message box that you can display from service applications. Change MB_OK to alter the buttons that appear. If you want the return value from MessageBoxEx change PopServiceBox into a function. 

procedure PopServiceBox(Title, Text: string);
begin
  MessageBoxEx(
    0, PChar(Text), PChar(Title),
    MB_OK or MB_TOPMOST or MB_SERVICE_NOTIFICATION, 0
  );
end;
