Force a TEdit to accept only numeric input

If got here by following a link such as http://delphidabbler.com/tips/999 (where 999 is a number) then you were re-directed because the tip is not yet available on the new delphidabbler.com website. Go back to the delphidabbler.com tips page

I've seen the tip to make a TEdit accept only numbers but there is an easier way to do it. This is the function I use in my applications: 

procedure SetEditNumeric(Handle: THandle);
begin
  SetWindowLong(Handle, GWL_STYLE,
    GetWindowLong(Handle, GWL_STYLE) or ES_NUMBER);
end;
