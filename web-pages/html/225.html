Double quote a string that has spaces

If got here by following a link such as http://delphidabbler.com/tips/999 (where 999 is a number) then you were re-directed because the tip is not yet available on the new delphidabbler.com website. Go back to the delphidabbler.com tips page

uses
  System.SysUtils...
...
var
  ...
  temp_str: String;
begin
...
{ double quote a string that has spaces }
if StrScan(PChar(temp_str), Chr($20)) <> nil then 
  temp_str := AnsiQuotedStr(temp_str, Chr($22));
...

$20 is the the space character; $22 is the double-quote character.

I know that QuoteSpacedString in the DelphiDabbler Code Snippets Databasedoes the same thing. The above if statement may be faster for strings that can only have $20 as a white space character such as a file path.

Tested on RAD Studio XE3.
